# Convoy Security Procedures

**Vehicle spacing:**
Minimum spacing: 50m between vehicles (prevents single IED or burst from hitting multiple). Maximum: visual contact maintained. In urban: reduce to 25m (prevent vehicle insertion into convoy).

**Speed:**
Move at speed appropriate to threat. Higher speed = less time in ambush kill zone. Consistent convoy speed prevents separation. Do not stop unless absolutely required.

**Formation:**
Lead vehicle (point): most experienced team, best eyes. Trail vehicle: provides rear security, assists disabled vehicles. Command vehicle: middle, protected position. Security vehicles: position based on threat (flank security if threat is flank, rear security if threat is rear).

**React to IED:**
Vehicles before and after blast: immediate 360 security. Treat casualties. Do not stop in kill zone longer than necessary. Establish security perimeter. Report.

**React to ambush:**
Last vehicle: turn 90° to block route. All vehicles: move forward through ambush at maximum speed (do not stop in kill zone). Disabled vehicle: crew moves to nearest vehicle and continues. Report immediately.

**Chokepoints:**
Reduce speed before chokepoints. Increase 360 security. Pass chokepoint quickly and resume spacing. One vehicle through at a time if necessary. Dismount scouts if threat level high.

**Communications:**
Convoy net on primary radio. All vehicles connected. Status checks every 15 minutes. Visual signals for communication failure.
