# Bounding Overwatch

**Purpose:** Move under fire using suppression and movement alternation. No one moves without fire support.

**Suppressing element:**
Establishes in covered/concealed position. Fires on threat to suppress while moving element bounds. Maintains continuous fire rate. Calls "SET" when bounding element reaches new position.

**Bounding element:**
Moves while suppressing element fires. Moves to next covered position forward of suppressing element. Takes cover. Calls "SET." Suppressing element then bounds.

**Positions:**
Alternating (leapfrog): one element always stationary providing fire. More controlled, slower.
Successive (caterpillar): both elements maintain alignment, one moves while other covers. Faster, requires more coordination.

**Bound distance:**
Limited by effective range of supporting fire and terrain. In open: 3–5 second rushes. In cover: bound from position to position. Never bound farther than supporting element can effectively engage the threat.

**Commands:**
"Element [color] – MOVE." Moving element responds: "Moving." When set: "Element [color] – SET." Supporting element: "Covering."

**Common error:** Moving element goes prone rather than to new covered position. Prone in open terrain provides no protection.
