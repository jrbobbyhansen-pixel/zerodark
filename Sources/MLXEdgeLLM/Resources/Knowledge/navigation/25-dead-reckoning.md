# Dead Reckoning – Navigating Without Reference

Dead reckoning: calculating current position from known starting point + direction + distance traveled.

**Formula:** Start position + (bearing × distance) = current position

**Inputs required:**
1. Known accurate start position
2. Magnetic bearing maintained throughout movement
3. Distance traveled (pace count or odometer)
4. Corrections for magnetic declination

**Execution:**
1. Record exact start position (GPS or grid reference)
2. Select and maintain compass bearing
3. Count paces or note distance
4. Plot on map: start – bearing line – mark distance along line
5. That point is dead reckoned position
6. Update continuously – errors accumulate over distance

**Error management:**
- Check against terrain features whenever possible – even a single identifiable hill lets you update
- Deviation: every degree off bearing = 1.7m error per 100m traveled. 10° off for 1km = 175m error.
- Regular stops to update position before error accumulates

**Night dead reckoning:**
More difficult – maintain precise bearing using compass held at chest level, red-light illuminated. Count paces meticulously. Move slower and accept reduced distance per hour.

**Sea/open terrain:**
Greatest DR use – no terrain features. Plot every leg. Mark timed positions. Calculate drift (wind/current effect) and compensate.
