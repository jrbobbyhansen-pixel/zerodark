# Return to Friendly Forces

**Protocol:** Never rush toward friendly positions – identification at distance prevents fratricide.

**Approach sequence:** 1. Gain attention from max safe distance (voice, signal mirror, bright color) 2. Keep hands visible 3. Move slowly and deliberately 4. State identity clearly and repeatedly 5. Follow all instructions exactly.

**Authentication:** Friendly forces will challenge. Respond with correct password. If unknown: stop, state identity, request authentication against ISOPREP. Be patient – authentication protects both sides.

**Night approach:** Use radio to announce approach. Request light signal from position. Move toward signal only. Don't approach positions that don't respond.

**Post-recovery:** Expect immediate debrief. Provide complete debrief – information may save others. Medical evaluation regardless of apparent health.

**Recovery assistance:** Leave clear signals at last position: direction of travel marked, messages under rocks or secured to vegetation. Standard signals increase successful recovery.
