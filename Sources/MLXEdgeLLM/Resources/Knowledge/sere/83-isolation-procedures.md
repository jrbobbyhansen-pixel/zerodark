# Isolated Personnel Procedures

**Immediate actions:** Establish security (360° observation). Assess situation (enemy proximity, terrain, weather). Treat injuries. Establish communication plan.

**Priorities:** 1. Security (avoid re-capture) 2. Signal (communicate position) 3. Survival (sustain self) 4. Move (toward friendly territory if needed)

**Authentication:** Use bona fides when contacting potential friendlies. Never surrender to unknown without friendly status confirmation. Friendly forces authenticate you too.

**Signaling:** Primary: radio on emergency frequency. Alternate: signal mirror. Contingency: pyrotechnic. Emergency: ground-to-air signals.

**Moving toward friendly lines:** Follow E&E route if possible. If compromised: navigate toward friendly position or border. Move at night, lie up during day. Report position frequently if radio available.

**Behavior in controlled environment:** Comply with legal orders. Resist interrogation beyond Big Four. Build rapport with other personnel. Maintain mental and physical discipline. Observe everything.

**Return to friendly control:** Never rush toward friendly positions. Identify at distance before closing. Use established authentication. Announce yourself loudly before approach.
