# Land Navigation

## Compass Navigation

### Compass Parts
- Magnetic needle (points north)
- Orienting lines and orienting arrow
- Direction-of-travel arrow
- Bezel ring with degree markings

### Taking a Bearing
1. Place compass on map with edge connecting current position to destination
2. Rotate bezel until orienting lines align with map grid lines
3. Read bearing where direction-of-travel arrow points on bezel
4. Rotate bezel until magnetic needle aligns with orienting arrow

### Following a Bearing in Field
1. Hold compass flat, point direction-of-travel arrow forward
2. Rotate your body until magnetic needle aligns with orienting arrow
3. Walk in direction of travel arrow
4. Check frequently; small angle errors compound over distance

### Back Azimuth
- Reverse direction by adding 180° to bearing
- Useful for returning via same route

## Celestial Navigation

### Sun Position
- Sun rises east, sets west, reaches highest point south at solar noon
- In Northern Hemisphere, sun is always south of observer
- In Southern Hemisphere, sun is always north of observer

### Star Navigation
- Northern Hemisphere: find North Star (Polaris)
  - Line up Pointers in Big Dipper, follow to Polaris
  - Polaris is true north
- Southern Hemisphere: use Southern Cross
  - Extend long axis 4-5 times, that point is south
- Sirius (brightest star), Orion's Belt aid navigation

### Watch Navigation (Northern Hemisphere)
1. Point hour hand at sun
2. Bisect angle between hour hand and 12 o'clock
3. That direction is south

## Terrain Association

### Map-to-Terrain Matching
1. Identify prominent features on map (ridges, peaks, valleys)
2. Find same features in terrain
3. Estimate distance to features
4. Trace route forward based on recognized terrain

### Terrain Features
- Ridge: obvious linear high ground
- Valley: depression following water drainage
- Saddle: low point between two peaks
- Slope: angle of terrain

## Dead Reckoning

**Procedure**:
1. Start from known position
2. Record compass bearing
3. Estimate distance traveled (pace count, time, visual)
4. Plot new position based on bearing and distance
5. Error accumulates; check against terrain features

**Pace Counting**:
- Establish normal pace in training (~2.5 feet per step)
- Count every step or every 2 steps
- Adjust for terrain (uphill/downhill changes pace length)

## Route Planning

1. Identify destination on map
2. Mark waypoints en route (distinctive terrain)
3. Note bearings between waypoints
4. Estimate distances, calculate time needed
5. Plan for obstacles (rivers, cliffs, dense vegetation)

## Navigation Without Map/Compass

- Follow water (streams lead to larger water, civilization)
- Use sun/stars for direction
- Follow animal trails (often lead to water)
- Use landmarks as reference points
- Keep sun position consistent if no compass

## Error Recognition

Signs you're off course:
- Terrain doesn't match expected features
- Sun position inconsistent with time of day
- Consistent bearing errors accumulate
- Unexpected obstacles

**Recovery**:
- Stop and reassess position
- Use terrain association to locate yourself
- Adjust bearing based on current position
- Consider backtracking to last known position
