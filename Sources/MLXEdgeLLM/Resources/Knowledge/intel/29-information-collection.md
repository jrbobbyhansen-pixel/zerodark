# Intelligence Collection

## Source Reliability Codes

**A** (always reliable) – **F** (cannot be judged). Information credibility: **1** (confirmed by other sources) – **6** (cannot be judged).

## SALUTE Format

**Size / Activity / Location / Unit / Time / Equipment**

Use for all observations.

## Pattern Analysis

Single event = incident. Two events = coincidence. Three or more = pattern with predictive value.

## PACE Planning

**Primary / Alternate / Contingency / Emergency** communication methods. Establish all four before operations begin.

## OSINT (Open Source)

Social media posts often contain location data in metadata. Public records show property ownership and relationships. Business registrations list officers and addresses.
