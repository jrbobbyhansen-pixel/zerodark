# ZeroDark iOS App

Open in Xcode to build and run on device:

```bash
cd ~/Developer/ZeroDark
open Package.swift
```

In Xcode:
1. Select "ZeroDarkApp" target
2. Select your iOS device or simulator (iOS 17+)
3. Build and run

## Features Included

- ☢️ **Nuclear Demo** — Voice → AI → Tools → Speech pipeline
- 👁️ **Screen Agent** — AI that sees and acts (macOS only)
- 🧠 **Fine-Tuning** — Train LoRA adapters on-device
- 🎤 **Voice Pipeline** — Full STT/TTS integration
- 🔧 **22 Agentic Tools** — Calendar, Reminders, HomeKit, HealthKit
- 🌐 **MCP Server** — Let external AIs use your device's tools

## Requirements

- iOS 17+ / macOS 14+
- Xcode 15+
- ~5GB free storage (per model)

## First Run

The app will download models on first launch. Recommended starting model: Qwen3 0.6B (fastest).
